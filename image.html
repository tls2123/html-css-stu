URI: uniform resource identifier(식별자) - 기본적으로 어떠한 자원을 식별 할 수 있는 모든 방법을 나타내는 것 ->학생을 부르는 방법 URL: uniform resource Locator(위치) ->왼쪽으로 2번째, 앞으로 2번째 있는 (2,2)학생 나와 URN: uniform resource name(이름) ->이름나와 -(기본적으로 ) 그 방법 하나하나 URL설명에 관한 링크 이 강의에서는 URL만 나옴 절대 경로와 상대 경로는 모두 URL을 표현하는 방법 1.절대경로 경로를 찾아내는 방법 - 전체를 써주는것 ex- 서울시 강서구 마곡동 어디어디 몇동 몇호 ex) https://via.placeholder.com/200 2.상대경로 우리집 위 우리집 아래 등등 ex) https://via.placeholder.com/, //example.com/ 와 같은 경우는 아님 ex) /text/example/aa.jpg와 같이 폴더를 타고 들어가는 것을 의미한다. #상태 경로 지정 방법 / : 루트 . : 현재 위치 .. : 위의 위치
